# JavaScript Conditional Statements

## What are Conditional Statements?

**Conditional statements** allow your program to make decisions and execute different code based on whether a condition is true or false.

They help control the flow of a program and respond to different situations.

---

## 1. If Statement

- The `if` statement executes a block of code **only if** the condition is true.
- If the condition is false, the code inside `if` is skipped.

---

## 2. If...Else Statement

- The `if...else` statement executes **one block** of code if the condition is true, and **another block** if the condition is false.
- It allows your program to handle two possible outcomes.

---

## 3. Else If Statement

- The `else if` statement is used to test **multiple conditions**.
- It allows you to check a series of conditions one by one.
- Only the first true condition will execute.

---

## 4. Ternary Operator

- The **ternary operator** is a shorter syntax for `if...else`.
- It has three parts: `condition ? valueIfTrue : valueIfFalse`.
- Useful for simple conditional assignments or expressions.

---

Understanding conditionals is a key step toward mastering JavaScript logic.
