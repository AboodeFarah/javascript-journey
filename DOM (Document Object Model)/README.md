## What is the DOM?

The **DOM (Document Object Model)** is a structure that represents an HTML webpage as objects that JavaScript can access and control.

In simple terms:  
**The DOM allows JavaScript to interact with and change a webpage.**

---

## Why the DOM is Important

- Makes websites interactive
- Allows updating content without refreshing the page
- Lets JavaScript control HTML and CSS
- Used in all modern web applications

---

## Selecting Elements

JavaScript can find elements in a webpage using different methods.

Common methods:

- **getElementById** → Select element by ID  
- **getElementsByClassName** → Select elements by class  
- **getElementsByTagName** → Select elements by tag  
- **querySelector** → Select first matching element  
- **querySelectorAll** → Select all matching elements  

---

## Manipulating Content

JavaScript can change the content of elements.

Important properties:

- **textContent** → Changes text only  
- **innerHTML** → Changes HTML content  

---

## Adding and Removing Elements

JavaScript can create and delete elements dynamically.

Important methods:

- **createElement** → Create new element  
- **appendChild** → Add element  
- **removeChild** → Remove element  

---

## Modifying Attributes

Attributes give extra information to elements.

Common methods:

- **setAttribute** → Add or change attribute  
- **getAttribute** → Get attribute value  
- **removeAttribute** → Remove attribute  

---

## Modifying Styles

JavaScript can change the appearance of elements.

This is done using:

- **style property**

Examples of style changes:

- Color  
- Font size  
- Border  
- Background color  
- Width and height  

---

## Key Concept

**DOM = JavaScript controlling the webpage dynamically.**
