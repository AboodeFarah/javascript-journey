# ES6 (ECMAScript 2015)

**ES6**, also known as **ECMAScript 2015**, is a major update to the JavaScript language.  
It introduced many new features that make JavaScript **cleaner, easier to write, and more powerful**.

ES6 helps developers write **modern JavaScript code** with better structure and readability.

## ES6 Features

### 1. Template Literals

**Definition:**  
Template literals allow you to include **variables and expressions inside strings** using **backticks (` `)** and the `${}` syntax.

Benefits:
- Easy string interpolation
- Cleaner string formatting
- Supports multi-line strings

---

### 2. Destructuring

**Definition:**  
Destructuring allows you to **unpack values from arrays or properties from objects into distinct variables**.

Benefits:
- Cleaner syntax
- Less repetitive code
- Easier access to array and object values

---

### 3. Default Parameters

**Definition:**  
Default parameters allow you to **assign default values to function parameters** if no argument is provided when the function is called.

Benefits:
- Prevents undefined values
- Makes functions safer and easier to use

---

### 4. Spread and Rest Operators

**Definition:**

- **Spread Operator (`...`)**  
  Expands an **array or object into individual elements**.

- **Rest Parameter (`...`)**  
  Collects **multiple values into a single array parameter**.

Benefits:
- Makes working with arrays and objects easier
- Simplifies function arguments handling
- Useful for copying and merging data
